# Connect4

A small Connect4 game written in ruby. Play against an AI or your friends.

## Installation

Requires Ruby >= 2.4.0

Installing dependencies, in the root directory:

    $ gem install bundler
    $ bundle install

## Usage

Running the application:

    $ ruby lib/Connect4.rb
